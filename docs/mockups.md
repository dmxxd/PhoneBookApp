# Макеты приложения

---

## 1. Окно авторизации

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/login_mockup.png" width="500">
</p>

**Назначение:** Авторизация зарегистрированного пользователя.

**Состав элементов:**
1.   **Заголовок окна:** Вход в систему
2.  **Центральный заголовок:** Телефонная книга (крупный жирный шрифт)
3.  **Поля ввода:** Email, Пароль.
4.   **Опция:** Чекбокс «Показать пароль».
5.  **Кнопки:** Войти, Регистрация (с утолщенными черными рамками).

## 2. Окно регистрации

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/registration_mockup.png" width="500">
</p>

**Назначение:** Создание нового аккаунта пользователя.

**Состав элементов:**
1.   **Заголовок окна:** Регистрация
2.  **Центральный заголовок:** Создание аккаунта.
3.  **Поля ввода:** Имя (активное поле с курсором), Email, Пароль, Подтверждение.
4.   **Кнопки:** Зарегистрировать, Отмена (с утолщенными черными рамками).

---

## 3. Главное окно

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/main_mockup.png" width="700">
</p>

**Назначение:** Основной интерфейс для работы с телефонной книгой.

**Состав элементов:**
1.   **Заголовок окна:** Телефонная книга
2.   **Верхняя панель:** Поле поиска «Поиск...» и кнопка «Фильтр по тегам».
3.  **Панель меню (серая полоса):** Разделы «Настройки» и «Выход».
4.  **Левая боковая панель:** Кнопки навигации «Управление пользователями» и «Добавить контакт».
5.   **Центральная область:** Широкое пустое поле (предназначено для отображения списка контактов).

---

## 4. Окно контакта

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/contact_mockup.png" width="500">
</p>

**Назначение:** Форма для заполнения данных о новом контакте или редактирования существующего.

**Состав элементов:**
1.   **Заголовок окна:** Контакт.
2.   **Поля ввода:** Имя * (обязательное поле), Фамилия, Телефон, Email, Адрес.
3.   **Многострочное поле:** Заметка.
4.  **Группа тегов:** Блок с чекбоксами (Важное, Друзья, Работа, Семья).
5.  **Выбор даты:** Поле «День рождения» (со значением `29.06.2026` и кнопкой с выпадающей стрелкой для календаря).
6.   **Кнопки:** Сохранить, Отмена.

---

## 5. Окно управления пользователями (Администратор)

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/user_management_mockup.png" width="500">
</p>

**Назначение:** Интерфейс администратора для управления аккаунтами пользователей.

**Состав элементов:**
1.  **Заголовок окна:** Управление пользователями.
2.  **Таблица пользователей:** 
    2.1   Заголовки (синий фон, белый текст): Имя, Email, Роль, Статус, Зарегистрирован.
    2.2   Строка с данными (серый фон): Admin, admin@phonebo..., Admin, Активен, 29.06.2026.
3.  **Кнопки управления (первый ряд):** Заблокировать, Разблокировать, Сделать Admin, Сделать User, Удалить.
4.   **Кнопки управления (второй ряд):** Просмотреть контакты (выравнивание по левому краю), Обновить (выравнивание по правому краю).
