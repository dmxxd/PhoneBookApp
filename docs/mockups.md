# Макеты приложения

---

## 1. Окно авторизации

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/login_mockup.png" width="700">
</p>

**Назначение:** Авторизация зарегистрированного пользователя.

**Состав элементов:**
*   **Заголовок окна:** Вход в систему
*   **Центральный заголовок:** Телефонная книга (крупный жирный шрифт)
*   **Поля ввода:** Email, Пароль.
*   **Опция:** Чекбокс «Показать пароль».
*   **Кнопки:** Войти, Регистрация (с утолщенными черными рамками).

## 2. Окно регистрации

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/registration_mockup.png" width="700">
</p>

**Назначение:** Создание нового аккаунта пользователя.

**Состав элементов:**
*   **Заголовок окна:** Регистрация
*   **Центральный заголовок:** Создание аккаунта.
*   **Поля ввода:** Имя (активное поле с курсором), Email, Пароль, Подтверждение.
*   **Кнопки:** Зарегистрировать, Отмена (с утолщенными черными рамками).

## 3. Главное окно

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/main_mockup.png" width="900">
</p>

**Назначение:** Основной интерфейс для работы с телефонной книгой.

**Состав элементов:**
*   **Заголовок окна:** Телефонная книга
*   **Верхняя панель:** Поле поиска «Поиск...» и кнопка «Фильтр по тегам».
*   **Панель меню (серая полоса):** Разделы «Настройки» и «Выход».
*   **Левая боковая панель:** Кнопки навигации «Управление пользователями» и «Добавить контакт».
*   **Центральная область:** Широкое пустое поле (предназначено для отображения списка контактов).

## 4. Окно добавления/редактирования контакта

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/contact_mockup.png" width="700">
</p>

**Назначение:** Форма для заполнения данных о новом контакте или редактирования существующего.

**Состав элементов:**
*   **Заголовок окна:** Контакт.
*   **Поля ввода:** Имя * (обязательное поле), Фамилия, Телефон, Email, Адрес.
*   **Многострочное поле:** Заметка.
*   **Группа тегов:** Блок с чекбоксами (Важное, Друзья, Работа, Семья).
*   **Выбор даты:** Поле «День рождения» (со значением `29.06.2026` и кнопкой с выпадающей стрелкой для календаря).
*   **Кнопки:** Сохранить, Отмена.

## 5. Окно управления пользователями (Администратор)

<p align="center">
<img src="https://github.com/dmxxd/PhoneBookApp/blob/master/images/user_management_mockup.png" width="900">
</p>

**Назначение:** Интерфейс администратора для управления аккаунтами пользователей.

**Состав элементов:**
*   **Заголовок окна:** Управление пользователями.
*   **Таблица пользователей:** 
    *   Заголовки (синий фон, белый текст): Имя, Email, Роль, Статус, Зарегистрирован.
    *   Строка с данными (серый фон): Admin, admin@phonebo..., Admin, Активен, 29.06.2026.
*   **Кнопки управления (первый ряд):** Заблокировать, Разблокировать, Сделать Admin, Сделать User, Удалить.
*   **Кнопки управления (второй ряд):** Просмотреть контакты (выравнивание по левому краю), Обновить (выравнивание по правому краю).
